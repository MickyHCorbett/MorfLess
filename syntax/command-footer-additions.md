# FOOTER_ADDITIONS

The FOOTER_ADDITIONS command will add content to the footer just before the closing body tag.

For example:

    ///FOOTER:
    %%FOOTER_ADDITIONS::
    content={ <text> }:


## Subschematic Commands

None

## Keywords

content={ <any text based content> }:

The text can be script, html or anything you like.
