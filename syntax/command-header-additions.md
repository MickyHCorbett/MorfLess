# HEADER_ADDITIONS

The HEADER_ADDITIONS command will add content to the head just before the closing head tag

For example:

    ///HEADER:
    %%HEADER_ADDITIONS::
    content={ <text> }:


## Subschematic Commands

None

## Keywords

content={ <any text based content> }:

The text can be script, html or anything you like.
