# CONTENT

The CONTENT command will add content - text, html etc to any section.

For example: 

    ///HEADER:
    %%CONTENT::
    TEXT=[ <text, html etc> ]:
    
## Subschematic Commands 

- TEXT (no arguments)

The syntax within the "=[ ]:" can be any text or html.

## Keywords

None
